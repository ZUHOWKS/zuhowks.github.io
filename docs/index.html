Chicken coop

Some people need for their chicken coop an intelligent door able to tell the difference between a chicken and a predator. This project answer that need. With the collaboration of my friends, we realised a mechanically opening door with an intelligent on-board system which use an AI. I was in charge to develop the program and configure the Raspberry Pi (a programmable card). To resume, the chicken coop house has a camera, a movement sensor and electromagnetic magnets to hold the door. When a presence is detected, a checking session is launched to see if there is a chicken. If it's the case, the electromagnets are disabled and the door can be opened.

What's Asilux ?

Asilux is a modded Minecraft server offering players new gameplay ! Players find themselves on an island embarked on an adventure in the kingdom of Asilux, where 6 clans fight in a game of glory. By choosing their clan, they must develop it and keep it active so as not to lose the race in this faction game.

How it works ?

Currently, the server is closed. However, you can now use the ALauncher by downloading it from the Releases section. You will be able to use and test the mods available in the current version of Asilux.
